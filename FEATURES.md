# Poorchid Features & Usage Guide

## Overview
Poorchid is a web-based chord synthesizer inspired by the Orchid hardware. It offers rich chord voicings, multiple performance modes, and a full suite of effects. This guide covers the main features and how to use them.

---

## Main Features

### 1. Chord Engine
- **Root Selection:** Choose any root note (C, C#, D, etc.)
- **Chord Types:** Major, Minor, Diminished, Suspended
- **Extensions:** Add 6, 7, m7, maj7, 9, 11, 13, etc.
- **Voicing Dial:** Adjusts chord spread and inversion
- **Key Mode:** Lock to a musical key and scale

### 2. Performance Modes
- **Direct:** Play full chords instantly
- **Arpeggiator:** Notes played in sequence, with pattern and division controls
- **Strum:** Chord notes strummed in rapid succession
- **Pattern:** Chord notes played in preset rhythmic patterns (straight, offbeat, pulse, tresillo, clave, shuffle, waltz, funk)

### 3. Effects (FX)
- **Reverb:** Algorithmic space effect
- **Delay:** BPM-synced stereo delay
- **Chorus:** Stereo shimmer
- **Phaser:** Sweeping filter
- **Flanger:** Metallic sweep
- **Drive:** Soft saturation/distortion
- **Tremolo:** Volume modulation
- **Ensemble:** Multi-voice chorus
- **FX Lock:** Prevents FX changes when switching patches

### 4. Bass Engine
- **Bass On/Off:** Toggle independent bass note
- **Bass Modes:** Chords, Unison, Solo, Direct
- **Bass Voicing:** Octave and offset control
- **Bass Volume:** Independent level

### 5. Looper
- **Record, Play, Overdub:** Capture and layer chord progressions
- **Undo:** Remove last layer

### 6. UI Controls
- **Encoders:**
  - **Sound:** Select patch/sound
  - **Perform:** Change performance mode, division, or pattern
  - **FX:** Select effect and adjust level
- **Buttons:**
  - Chord type, extension, key, bass, looper, metronome

---

## How to Use Poorchid

### Getting Started
1. **Power On:** Click the power button to enable audio.
2. **Select Root & Chord:** Use the root and chord type buttons to choose your chord.
3. **Add Extensions:** Click extension buttons to add 7th, 9th, etc.
4. **Adjust Voicing:** Turn the voicing dial to spread or invert the chord.
5. **Key Mode:** Enable key mode to lock chords to a scale.

### Performance Modes
- **Direct:** Default mode, plays full chord.
- **Arp:** Click/turn the perform encoder to switch to arpeggiator. Adjust division and pattern.
- **Strum:** Select strum mode for guitar-like strumming.
- **Pattern:** Select pattern mode for rhythmic chord playback.

### Effects
- **Select FX:** Turn the FX encoder to cycle through effects.
- **Adjust Level:** Rotate to set effect depth (0-99).
- **Lock FX:** Hold FX encoder to lock settings when changing sounds.

### Bass
- **Enable Bass:** Click the bass button.
- **Change Mode:** Cycle bass modes for different behaviors.
- **Adjust Voicing/Volume:** Use bass controls for octave and level.

### Looper
- **Record:** Click record to start capturing chords.
- **Play/Overdub:** Click play to loop, record again to overdub.
- **Undo:** Remove last overdub layer.

### Other Controls
- **Metronome:** Toggle for click track.
- **Tap Tempo:** Tap to set BPM.
- **Volume:** Adjust master and bass volume.

---

## Tips
- All controls are available via mouse, touch, or MIDI.
- State changes are reflected instantly in sound and UI.
- Use FX lock to keep your favorite effect settings when switching patches.
